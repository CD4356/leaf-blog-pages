# leaf-blog-pages
基于semantic ui前端ui框架快速搭建的个人博客页面，里面集成了markdown编辑器、particles.js炫酷粒子背景、prism代码高亮、tocbot目录生成、animates动画效果、waypoints滚动侦测、jquery.scrollTo平滑滚动、qrcode二维码等使用的前端插件，对于有意于搭建个人网站的前端小白来说是一个挺不错的学习资源
